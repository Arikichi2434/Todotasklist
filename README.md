# Todo Task List

HTML・CSS・JavaScriptの学習用に作ったシンプルなTodoアプリです。

## 使い方

`index.html` をブラウザで開くと動きます。

## 学べるポイント

- HTML: フォーム、ボタン、リスト、アクセシビリティ用ラベル
- CSS: レイアウト、レスポンシブ対応、状態ごとの見た目
- JavaScript: DOM操作、イベント処理、配列操作、localStorage保存

## 問題集

コードを理解するための穴埋め問題と解説は [problem-set.md](problem-set.md) にあります。

## 主な機能

- タスクの追加
- 完了・未完了の切り替え
- タスクの削除
- すべて・未完了・完了のフィルター
- 完了済みタスクの一括削除
- ブラウザへの自動保存
